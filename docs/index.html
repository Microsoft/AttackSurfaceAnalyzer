Attack Surface Analyzer

This documentation covers the API of the Attack Surface Analyzer library.

Getting Attack Surface Analyzer

Attack Surface Analyzer is distributed via our GitHub releases page and via NuGet.

Detailed information on how to use Attack Surface Analyzer can be found on our wiki.

Installation

Attack Surface Analyzer runs on Windows, Linux, and MacOS, and is built using .NET Core.

Packages are available on our releases page as compressed archives. And available on NuGet with dotnet tool install -g Microsoft.CST.AttackSurfaceAnalyzer.CLI and the library is Microsoft.CST.AttackSurfaceAnalyzer.

Building

To build Attack Surface Analyzer, see BUILD.
